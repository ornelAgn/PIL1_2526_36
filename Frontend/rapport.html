3. Description de la Conception

Architecture Logicielle

L'application adopte une architecture Client-Serveur modulaire :

  • Frontend : HTML, CSS, JavaScript (templates Django).
  • Backend : Développé en Python avec le framework Django (version 6.0.6).

Conception de la Base de Données (Modèle Relationnel)

La base de données relationnelle a été implémentée sous MySQL. Les mots de passe sont stockés de manière sécurisée grâce au hachage intégré de Django.

Les principales tables incluent : Profile, Skill, MentorSkill, MenteeSkill, Conversation, ConversationMember, Message, MentorshipSession, Review, Badge, UserBadge.

4. Instructions de Déploiement

Prérequis

  • Python 3.11 ou supérieur installé (testé avec 3.14.2)
  • SGBD MySQL configuré et actif
  • Git installé
  • Un environnement virtuel Python recommandé (venv)

Étape 1 : Cloner le projet

git clone https://github.com/ornelAgn/PIL_2526_36.git
cd PIL_2526_36

Étape 2 : Créer et activer un environnement virtuel

python -m venv venv
# Windows
venv\Scripts\activate
# Linux / macOS
source venv/bin/activate

Étape 3 : Installer les dépendances Python

pip install -r requirements.txt

Le fichier requirements.txt contient notamment :

  • Django==6.0.6
  • mysqlclient==2.2.0 (ou PyMySQL==1.2.0 selon configuration)
  • django-crispy-forms (si utilisé)
  • autres paquets mineurs (Pillow, etc.)

Étape 4 : Configurer la Base de Données

  1. Créez une base de données nommée projet_db dans MySQL.
  2. Importez la structure fournie : 
    mysql -u root -p projet_db < database/schema.sql
  3. Adaptez les paramètres de connexion dans backend/backend/settings.py (USER, PASSWORD, etc.).

Étape 5 : Appliquer les migrations Django

cd backend
python manage.py migrate

Étape 6 : Lancer le serveur de développement

python manage.py runserver

L'application sera accessible sur http://127.0.0.1:8000.

6. Manuel d'Utilisation

1. Gestion de compte

  • Inscription : Remplissez le formulaire avec vos informations personnelles et académiques, puis validez.
  • Connexion : Utilisez votre nom d’utilisateur et votre mot de passe pour accéder à votre espace.

2. Tableau de bord et recherche de mentors

  • Après connexion, le tableau de bord affiche les mentors disponibles et vos sessions à venir.
  • Utilisez la barre de recherche pour trouver un mentor par compétence, filière ou nom.

3. Matching intelligent

  • La page « Matching » classe les mentors selon un score de compatibilité basé sur vos compétences recherchées, vos disponibilités, votre filière et l’expérience du mentor.

4. Messagerie instantanée

  • Accédez à la messagerie pour voir vos conversations.
  • Ouvrez une conversation pour envoyer et recevoir des messages en temps réel.
