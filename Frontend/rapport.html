3. Description de la Conception

Architecture Logicielle

L'application adopte une architecture Client-Serveur modulaire :

  • Frontend : Technologies de base HTML, CSS, JavaScript .
  • Backend : Développé en Python avec le framework (Flask).

La base de données relationnelle a été implémentée sous MySQL. Les mots de passe y sont stockés de manière sécurisée grâce à une fonction de hachage.

4. Instructions de Déploiement

Suivez ces étapes pour installer et exécuter l'application localement sur votre machine :

Prérequis

  • Python 3.11.9 installé
  • SGBD MySQL configuré et actif
  • Git installé
  • Création d'un dépot GitHub

Étape 1 : Cloner le projet

git clone https://github.com/.ornelAgn/PIL_2526_36.git cd PIL1_2526_36

Étape 2 : Installer les dépendances Python

pip install -r requirements.txt

Étape 3 : Configurer la Base de Données

  1. Créez une base de données nommée ifri_mentorlink dans votre SGBD.
  2. Importez le fichier de structure fourni à la racine : 
    mysql -u root -p ifri_mentorlink <

Étape 4 : Lancer le serveur

Exécutez la commande suivante pour lancer l'application backend :

 python app.py ou python -m flask run

L'application sera accessible dans votre navigateur à l'adresse : http://127.0.0.1:5000

5. Manuel d'Utilisation

1. Gestion de compte

  • Inscription : Allez sur la page d'inscription, remplissez vos informations de base (Nom, Prénom, Email, Téléphone, etc.), puis définissez vos lacunes et vos lacunes points forts.
  • Connexion : Utilisez votre email ou numéro de téléphone unique et votre mot de passe pour accéder à votre espace.

2. Fonctionnalité de Mentorat & Matching

  • Publier une offre/demande : Remplissez le formulaire en spécifiant la matière concernée, vos disponibilités et le format (En ligne / Présentiel).
  • Voir les suggestions (Matching) : Cliquez sur l'onglet "Mes suggestions de matching". L'algorithme calculera automatiquement les profils les plus compatibles en fonction de vos compétences communes, horaires partagés et filières d'études.

3. Messagerie instantanée

  • Ouvrez l'onglet "Messagerie" pour voir la liste de vos conversations en cours.
  • Sélectionnez un profil "matché" pour lui envoyer un message en temps réel.
